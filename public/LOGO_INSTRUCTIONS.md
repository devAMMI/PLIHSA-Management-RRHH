# Instrucciones para el Logo PLIHSA

## Archivo Requerido
**Nombre:** `Logo_PLIHSA_BLUE.png`
**Ubicación:** `/public/Logo_PLIHSA_BLUE.png`

## Descripción
El logo debe ser el archivo LOGO_PLIHSA_AZUL_(1).png proporcionado - un óvalo azul con el texto "PLIHSA" en blanco.

## Uso
Este logo se utiliza en:
- Formulario de Definición de Metas Administrativas
- Formulario de Definición de Metas Operativas
- Vista/Edición/Impresión de Definiciones de Metas
- Pantalla principal de Definición de Metas

## Formato
- Formato: PNG con fondo transparente o sólido
- Dimensiones recomendadas: Mínimo 800px de ancho
- El logo será escalado automáticamente para ajustarse a un máximo de 180px

## Colocación
1. Guarda el archivo `LOGO_PLIHSA_AZUL_(1).png`
2. Renómbralo a `Logo_PLIHSA_BLUE.png`
3. Colócalo en la carpeta `/public/` del proyecto
4. El sistema automáticamente lo cargará en todos los formularios
